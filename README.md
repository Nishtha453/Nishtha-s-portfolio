# 🖥️ Terminal Portfolio — Nishtha Gupta
 
A terminal-style interactive portfolio. No carousels, no hero sections — navigate my work the way a developer would, through a command-line interface.
 
**Live:** [nishtha-gupt-portfolio-website.netlify.app](https://nishtha-gupt-portfolio-website.netlify.app)  
**Repo:** [github.com/Nishtha453/Nishtha-s-portfolio](https://github.com/Nishtha453/Nishtha-s-portfolio)
 
---
 
## ⚡ Commands
 
Type any of these into the terminal prompt on the live site:
 
| Command        | Description                        |
|----------------|------------------------------------|
| `help`         | List all available commands        |
| `ls`           | Browse sections                    |
| `about`        | Who I am                           |
| `projects`     | Things I've built                  |
| `research`     | Papers in progress                 |
| `skills`       | Technical stack                    |
| `achievements` | Awards and credentials             |
| `contact`      | Reach me                           |
| `clear`        | Clear the terminal                 |

```
├── index.html        # entry point + terminal UI

```

## 🚀 Run Locally
 
```bash
# Clone the repo
git clone https://github.com/Nishtha453/Nishtha-s-portfolio.git
cd Nishtha-s-portfolio
 
# Open directly — no install, no build step
open index.html
 
# Or serve locally
npx serve .
```
